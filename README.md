# Security+ SY0-701 Practice Exams

A study tool for the CompTIA Security+ SY0-701 certification exam featuring three full practice quizzes with 85 multiple-choice questions each.

## Features
- **Quiz A, B, and C** — 85 questions each
- **Practice mode** — check answers as you go
- **Timed Exam mode** — simulate real test conditions
- **Shuffle** — randomize question order
- **Instant feedback** — see correct/incorrect answers immediately
- **Score tracking** — progress bar, final grade, and full review

## Running Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **Add New → Project**
4. Import this repo
5. Click **Deploy**

Done — you'll get a free live URL.
